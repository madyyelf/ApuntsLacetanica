2. Idea inicial

Realitzar una Jam per a crear la nova web de http://infla.cat.

3. Bases

3.1. Jam de 09:00h a 21:00h a la 3.3

  • D'aquesta forma donem les mateixes oportunitats a matís/tardes. Poden si volen fer les 12 hores seguides o tan sols utilitzar les del seu torn.
  • Caldrà reservar l'aula 3.3. tot el dia (modificant horaris si cal) i professorat que l'atengui. Especialment horari de migdia.

3.2. Objectiu: Web estàtica amb HTML i CSS

  • En ser una web sense CMS ni elements programació per part del servidor, si la web es queda sense manteniment no correm el risc que ens la hackejin.
  • En utilitzar tan sols HTML i CSS anivellem tant GM i GS, tots els alumnes participaran amb coneixements que tenen per igual.
  • Responsive
  • Tots els elements amb llicència d'ús.

3.3. Temàtica explicada el mateix dia

  • Cal explicar-ho el mateix dia per a evitar que algú ho comenci a treballar abans.
  • La temàtica serà fer una web per a http://infla.cat.
  • Un apartat per cicle explicant el perfil professional, marc horari, mòduls…
  • Certificats addicionals?
  • Logo (nou?)

3.4. Premis

  • 1r premi monetari per incentivar la participació.
  • 1r, 2n i 3r premis amb samarreta de la Web Jam d'Infla 24-25.

3.5. Promoció

3.5.1. Interna

  • Ambient per "passar l'estona" (música, quelcom per picar, …)
  • Projecció de pantalles mentre es treballa.

3.5.2. Externa

  • Retransmissió per Twitch.
  • Edició posterior de vídeo.
  • Publicació de totes les web candidates en la web de infla.
